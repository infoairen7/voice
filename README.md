# 逆再生ボイススタジオ / Reverse Voice Party

録って、逆にして、変な声で笑おう。

配信中にその場でマイク録音し、ワンクリックで逆再生＆ボイスチェンジできるブラウザツールです。VTuber・個人配信者・雑談配信者向けの「配信ネタ用」ツールとして作られています。

- サーバー不要。HTML / CSS / JavaScript だけで動きます
- 音声はすべて端末（ブラウザ）内で処理され、外部には一切送信されません
- GitHub Pages にそのまま置けば公開できます

## 主な機能

- マイク録音（最大15秒・自動ストップ・3秒カウントダウンあり）
- 通常再生 / 逆再生（AudioBuffer のチャンネルデータを reverse して再生）
- ボイスチェンジャー7モード（ノーマル / チップマンク / 魔王 / ロボット / ラジオ / エコー / ホラー）
- スピードスライダー（ピッチも一緒に変わります）
- お題（サンプルテキスト）のカード表示＆ランダム表示
- マイク入力の音量メーター＆波形風ビジュアライザー
- 録り直しボタン
- WAV保存（普通の声 / 逆再生 / 変な声で逆再生を選択可能。エフェクトごと書き出します）
- OBS等に小さく載せたい時の「ミニモード」

## 使い方

1. `index.html` をブラウザで開く（またはGitHub Pagesで公開したURLを開く）
2. 「録音スタート」を押す → 初回はブラウザがマイクの許可を聞いてくるので「許可」を選ぶ
3. お題を読み上げる（「お題をランダム」でネタが出ます）
4. 「録音ストップ」を押す（15秒で自動的に止まります）
5. 「逆再生する」で逆再生。「変な声で逆再生」で選択中のボイスモード付きで逆再生
6. ボイスモードは右のパネルから切り替え。「🎲 おまかせ」でランダム選択
7. 気に入ったら「音声を保存」でWAVとしてダウンロード
8. 「録り直す」で今の録音を捨てて、もう一度録音できます

再生中にもう一度同じボタンを押すと停止します。

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作る（例：`reverse-voice-party`）
2. この4ファイルをリポジトリのルートにアップロードする
3. リポジトリの **Settings → Pages** を開く
4. 「Build and deployment」の Source を **Deploy from a branch** にする
5. Branch を `main`（フォルダは `/root`）にして **Save**
6. 数分待つと `https://ユーザー名.github.io/reverse-voice-party/` で公開されます

※ マイク（getUserMedia）は **HTTPSでしか動きません**。GitHub PagesはHTTPSなのでそのまま使えます。ローカルで試す場合は `file://` でも多くのブラウザで動きますが、動かない場合は簡易サーバー（例：`python3 -m http.server`）経由で `http://localhost` から開いてください（localhostは特別に許可されています）。

## OBSで使う場合の注意

**おすすめの構成：ふつうのブラウザで開いて、OBSでキャプチャする**

1. ChromeなどでこのツールのURLを開き、マイクを許可する
2. OBSに「ウィンドウキャプチャ」でそのブラウザを追加する
3. 音は以下のどちらかで乗せる
   - 「アプリケーション音声キャプチャ」でブラウザの音を拾う
   - または「デスクトップ音声」を有効にする

**OBSの「ブラウザソース」に直接入れる場合の注意**

- OBSのブラウザソースはマイクの許可ダイアログが出ないため、**録音ができないことがあります**
- 見た目（ビジュアライザーやお題表示）だけをブラウザソースで映し、操作と録音は別のブラウザで行う、という使い分けもできます
- 画面が狭い時はヘッダーの「ミニモード」ボタンでコンパクト表示になります

## マイク権限について

- 初回に「録音スタート」を押すと、ブラウザがマイクの使用許可を求めます
- 一度ブロックしてしまった場合は、アドレスバー左のカギ（またはマイク）アイコン → サイトの設定 → マイクを「許可」に変更してからページを再読み込みしてください
- マイクは録音とメーター表示のためだけに使い、音声データはページを閉じると消えます

## 音が出ない・録音できない時の確認ポイント

| 症状 | 確認すること |
| --- | --- |
| 録音スタートを押しても反応しない | マイク許可がブロックされていないか（上記「マイク権限について」参照） |
| 「対応していない」と表示される | 最新のChrome / Edge / Safari / Firefoxで開き直す |
| 再生ボタンが押せない | まだ録音していない状態です。先に録音してください |
| スマホで音が鳴らない | マナーモード（消音スイッチ）を解除。ボタンを押した操作で再生される設計ですが、音量がゼロになっていないかも確認 |
| メーターが動かない | 一度「録音スタート」を押してマイクを有効にする。OSのサウンド設定で正しい入力デバイスが選ばれているか確認 |
| OBSに音が乗らない | OBS側で「アプリケーション音声キャプチャ」または「デスクトップ音声」が有効か確認 |
| 声が小さい/割れる | マイクとの距離を調整。OSの入力レベルも確認 |

## 技術メモ

- 録音：`MediaRecorder` API（webm/opus等、ブラウザが対応する形式を自動選択）
- 逆再生：録音データを `decodeAudioData` で `AudioBuffer` にし、各チャンネルの `Float32Array` を `reverse()` した逆向きバッファを作成して再生
- ボイスチェンジ：`playbackRate` / `BiquadFilterNode` / `DelayNode` / `GainNode` / `WaveShaperNode` / オシレーターによるリングモジュレーション・トレモロ の組み合わせ
- 保存：`OfflineAudioContext` でエフェクトごとレンダリングし、16bit PCM のWAVに変換してダウンロード
- 外部ライブラリなし（Webフォントのみ Google Fonts を利用。オフラインでもシステムフォントで動作します）
